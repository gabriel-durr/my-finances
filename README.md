> <h3> Getting start : </h3>
<br/>

<div>

1. <b>Clone the project</b>

```bash
  $ git clone https://github.com/gabriel-durr/my-finances.git  &&  cd my-finances
```



2. <b>Install dependencies</b> 

```bash
  $ yarn 
```


3. <b>Start the application</b> 

```bash
  $ yarn server
  $ yarn start
```
